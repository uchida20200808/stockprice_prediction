PythonのTensorflowを使って株価の予測をするプログラム

yahoo financeから株価のデータを取得する。

取得したデータの80%を使って機械学習をする。

残りの20%のデータから株価の予測を立てる。

予測方法は過去60日の終値から翌61日目の終値を計算するトレーニングを機械学習で行う。

最後に明日の株価を予測するプログラムが紹介されている。
